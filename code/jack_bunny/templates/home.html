This is inspired by the bunnylol search engine at facebook. There's an open version currently available, released by Charlie Cheever at bunny1 but it hasn't been updated in a while so I decided to release my own. I included more commands that I personally found to be really useful and I hope you enjoy it :)

Installing on Google Chrome

  • Open settings. Scroll down to the section where it says "Search engine"
  • Click 'Manage search engines'. Next to 'Other search engines', click the ADD button
  • Name the 'Search engine' and 'Keyword' field whatever you want. Inside the URL field, paste this: http://bunny.jackjyang.com/q/?query=%s
  • Set this search engine to your default
  • Type 'help' into your address bar to see which commands are currently available
